How it works

1

Run one command

Paste the install command into your terminal. The script detects your OS, architecture, and package manager automatically.

2

Pick your tools

Use the interactive picker or load a preset. Dependencies are resolved automatically — Node installs before npm-based tools.

3

Start building

Everything installs in parallel phases. Already-installed tools are skipped. You get a summary with next steps for auth setup.

Share your stack

Drop a bundledownload.json in your repo. New team members run one command and get your exact toolset.

Works with local files, URLs, or auto-detected from the current directory.

{
  "name": "my-team",
  "tools": [
    "node", "git", "gh",
    "vercel", "supabase",
    "claude-code"
  ],
  "node_version": "lts"
}
